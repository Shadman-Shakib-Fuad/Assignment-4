# Job Interview Card System

## 📌 Project Overview

This project is a simple Job Interview Card System built using HTML, CSS, and JavaScript. It displays interview cards in a structured layout and allows users to mark candidates as "Interviewed" or "Rejected" dynamically.

---

## 🚀 Features

* Display interview cards in a single-column layout (one card per row)
* Interactive buttons for:

  * Interview → turns into a green "Interviewed" button
  * Reject → turns into a red "Rejected" button
* Smooth hover effects on cards
* Clean and responsive UI design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 📂 How to Run the Project

1. Download or clone the project files
2. Open the folder
3. Double-click on the `index.html` file
   OR
   Open it using any browser (Chrome, Edge, Firefox)
